========
AVERAGES
========

------------------------------------------------------------

BASIC DEFINITION
----------------

Average

    = Sum of Observations
      ----------------------
      Number of Observations


If total sum = S
and number of items = n,

    Average

    = S / n


------------------------------------------------------------

1) FINDING TOTAL SUM
--------------------

If Average = A
and number of items = n,

Total Sum

    = A × n


------------------------------------------------------------

2) WHEN ONE NEW ITEM IS ADDED
-----------------------------

New Average

    = (Old Sum + New Value)
      -------------------------
      (n + 1)


------------------------------------------------------------

3) WHEN ONE ITEM IS REMOVED
---------------------------

New Average

    = (Old Sum − Removed Value)
      ------------------------------
      (n − 1)


------------------------------------------------------------

4) CHANGE IN AVERAGE
--------------------

If average increases by x,

Increase in Total Sum

    = n × x


If average decreases by x,

Decrease in Total Sum

    = n × x


------------------------------------------------------------

5) COMBINED AVERAGE
-------------------

If two groups have:

First group:

    Number = n₁
    Average = A₁


Second group:

    Number = n₂
    Average = A₂


Combined Average

    = (n₁A₁ + n₂A₂)
      -------------------
        (n₁ + n₂)


------------------------------------------------------------

6) WEIGHTED AVERAGE
-------------------

Weighted Average

    = (w₁x₁ + w₂x₂ + w₃x₃ + ...)
      --------------------------------
        (w₁ + w₂ + w₃ + ...)


Where,

    w = weights
    x = values


------------------------------------------------------------

7) SUCCESSIVE CHANGE IN AVERAGE
-------------------------------

If average of n numbers is A
and each number increases by k,

New Average

    = A + k


If each number decreases by k,

New Average

    = A − k


------------------------------------------------------------

8) AVERAGE SPEED (IMPORTANT)
----------------------------

If equal distances are covered at speeds S₁ and S₂,

Average Speed

    = 2 × S₁ × S₂
      ----------------
      (S₁ + S₂)


If different distances,

Average Speed

    = Total Distance
      ----------------
        Total Time


------------------------------------------------------------

IMPORTANT SHORT NOTES
---------------------

• Average = Sum / Number  
• If average increases → total sum increases  
• Combined average is weighted average  
• Equal distance → use harmonic mean formula  
• Adding same number to all → average shifts by same number
