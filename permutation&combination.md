===========================
PERMUTATION AND COMBINATION
===========================

------------------------------------------------------------

BASIC DEFINITIONS
-----------------

Permutation

    Arrangement of objects
    (Order matters)


Combination

    Selection of objects
    (Order does NOT matter)


------------------------------------------------------------

FACTORIAL
---------

n!

    = n × (n − 1) × (n − 2) × .... × 1


Special cases:

    0! = 1
    1! = 1


------------------------------------------------------------

PERMUTATION FORMULA
-------------------

Number of arrangements of r objects
from n distinct objects


nPr

    = n!
      ------------
      (n − r)!


When all objects are arranged,

nPn

    = n!


------------------------------------------------------------

PERMUTATION WITH REPEATITION
----------------------------

If total objects = n
and some are identical,

Number of arrangements

    = n!
      -------------------------
      p! × q! × r! ...


Where,

    p, q, r = identical objects


------------------------------------------------------------

CIRCULAR PERMUTATION
--------------------

Number of arrangements in a circle


    = (n − 1)!


If clockwise and anticlockwise
are considered same,

    = (n − 1)! / 2


------------------------------------------------------------

COMBINATION FORMULA
-------------------

Number of ways to select r objects
from n distinct objects


nCr

    = n!
      -------------------
      r! (n − r)!


------------------------------------------------------------

RELATION BETWEEN nPr AND nCr
----------------------------

nPr

    = nCr × r!


------------------------------------------------------------

IMPORTANT IDENTITIES
--------------------

nCr

    = nC(n − r)


nC0

    = 1


nCn

    = 1


------------------------------------------------------------

TOTAL NUMBER OF SUBSETS
-----------------------

If total elements = n,

Number of subsets

    = 2^n


Number of proper subsets

    = 2^n − 1


------------------------------------------------------------

IMPORTANT SHORT NOTES
---------------------

• Use Permutation when arrangement matters  
• Use Combination when selection matters  
• Circular seating → use (n − 1)!  
• Identical objects → divide by factorial  
• Remember symmetry rule: nCr = nC(n − r)  

------------------------------------------------------------

END OF NOTES
