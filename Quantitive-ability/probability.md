===========
PROBABILITY
===========

------------------------------------------------------------

BASIC DEFINITION
----------------

Probability of an Event

    = Number of Favourable Outcomes
      --------------------------------
      Total Number of Possible Outcomes


If total outcomes = n
and favourable outcomes = m,

    P(E)

    = m / n


------------------------------------------------------------

RANGE OF PROBABILITY
--------------------

0  ≤  P(E)  ≤  1


Impossible Event

    P(E) = 0


Certain Event

    P(E) = 1


------------------------------------------------------------

COMPLEMENTARY EVENT
-------------------

Probability of NOT occurring

    P(Ē)

    = 1 − P(E)


------------------------------------------------------------

ADDITION RULE
-------------

If A and B are two events,

P(A or B)

    = P(A) + P(B) − P(A ∩ B)


If A and B are mutually exclusive,

    P(A ∩ B) = 0

So,

    P(A or B)

    = P(A) + P(B)


------------------------------------------------------------

MULTIPLICATION RULE
-------------------

If A and B are independent events,

P(A and B)

    = P(A) × P(B)


If events are dependent,

P(A and B)

    = P(A) × P(B given A)


------------------------------------------------------------

CONDITIONAL PROBABILITY
-----------------------

P(A given B)

    = P(A ∩ B)
      ----------
        P(B)


------------------------------------------------------------

TOTAL PROBABILITY RULE
----------------------

If B1, B2, B3 are mutually exclusive and exhaustive events,

P(A)

    = P(A|B1)P(B1)
    + P(A|B2)P(B2)
    + P(A|B3)P(B3)


------------------------------------------------------------

BAYES' THEOREM
--------------

P(B1|A)

    = P(A|B1) P(B1)
      -------------------------------
      P(A|B1)P(B1) + P(A|B2)P(B2)


------------------------------------------------------------

PERMUTATION (IMPORTANT FOR PROBABILITY)
---------------------------------------

nPr

    = n!
      ------
      (n − r)!


------------------------------------------------------------

COMBINATION (VERY IMPORTANT)
----------------------------

nCr

    = n!
      ------------
      r! (n − r)!


Used when order does NOT matter.
