=============================
       PERCENTAGE
=============================
x% of y = (x/100) × y

---------------------------------
   Increase formula:
--------------------------------
Final = Original × (1 + r/100)

// If something increases by r%, it becomes:

Original + r% of Original

= Original + (r/100 × Original)
= Original (1 + r/100)

 Example:
Salary = 40,000
Increase = 10%

Final = 40000 × (1 + 10/100)
= 40000 × 1.1
= 44,000

--------------------------------------
   Decrease formula:
--------------------------------------
Final = Original × (1 − r/100)

Original − r% of Original
= Original − (r/100 × Original)
= Original (1 − r/100)

 Example:

Price = 800
Decrease = 20%

Final = 800 × (1 − 20/100)
= 800 × 0.8
= 640


------------------------------------------------------------

1) SUCCESSIVE CHANGE FORMULA
------------------------------------------------------------

If a quantity changes by a% and then by b%,

Final Value

    = Original Value × (1 + a/100) × (1 + b/100)


Expanding:

    = 1 + a/100 + b/100 + ab/10000


Therefore,

    Net Percentage Change

    = a + b + ab/100


------------------------------------------------------------

2) PERCENTAGE CHANGE
------------------------------------------------------------

Percentage Change

    = (New Value − Old Value) / Old Value × 100


------------------------------------------------------------

3) REVERSE PERCENTAGE
------------------------------------------------------------

If value increases by r%,

    Original Value

    = Final Value / (1 + r/100)


If value decreases by r%,

    Original Value

    = Final Value / (1 − r/100)


------------------------------------------------------------

4) POPULATION GROWTH
------------------------------------------------------------

Population after n years

    = P × (1 + r/100)^n


Where,

    P = Initial Population
    r = Growth Rate
    n = Number of Years


------------------------------------------------------------

5) COMPOUND INTEREST
------------------------------------------------------------

Amount (A)

    = P × (1 + r/100)^n


Where,

    P = Principal
    r = Rate of Interest
    n = Time (Years)


------------------------------------------------------------

6) IF A IS x% MORE THAN B
------------------------------------------------------------

    A = B × (1 + x/100)


If A is x% less than B,

    A = B × (1 − x/100)


------------------------------------------------------------

7) COMPARISON FORMULAS
------------------------------------------------------------

If A is x% more than B,
then B is less than A by:

    x / (100 + x) × 100


If A is x% less than B,
then B is more than A by:

    x / (100 − x) × 100


------------------------------------------------------------

8) PRICE–QUANTITY (PRODUCT CONSTANT)
------------------------------------------------------------

If price increases by x%,
required decrease in quantity:

    x / (100 + x) × 100


------------------------------------------------------------

9) MARKS PERCENTAGE
------------------------------------------------------------

Percentage

    = Marks Obtained / Total Marks × 100


------------------------------------------------------------

10) PROFIT & LOSS
------------------------------------------------------------

Profit %

    = Profit / Cost Price × 100


Loss %

    = Loss / Cost Price × 100


------------------------------------------------------------

11) SUCCESSIVE DISCOUNT
------------------------------------------------------------

If discounts are a% and b%,

Net Discount

    = a + b − ab/100


------------------------------------------------------------

12) INCREASE FOLLOWED BY SAME DECREASE
------------------------------------------------------------

If a number increases by x%
and then decreases by x%,

Net Loss

    = x² / 100


------------------------------------------------------------

IMPORTANT FRACTION TO PERCENTAGE
------------------------------------------------------------

1/2  = 50%
1/3  = 33.33%
1/4  = 25%
1/5  = 20%
1/6  = 16.67%
1/8  = 12.5%
1/9  = 11.11%
1/10 = 10%
