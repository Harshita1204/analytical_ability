==============
TIME AND WORK
==============

------------------------------------------------------------

BASIC CONCEPT
-------------

If a person completes a work in T days,

Work done in 1 day

    = 1 / T


Total Work is generally assumed as 1 unit.

------------------------------------------------------------

1) INDIVIDUAL WORK FORMULA
--------------------------

If A completes work in a days,

    A's 1 day work

    = 1 / a


If B completes work in b days,

    B's 1 day work

    = 1 / b


------------------------------------------------------------

2) TOGETHER WORK FORMULA
------------------------

If A and B work together,

Total 1 day work

    = 1/a + 1/b


Time taken together

    = 1 / (1/a + 1/b)


Simplified form:

    = ab / (a + b)


------------------------------------------------------------

3) THREE PERSONS WORKING TOGETHER
---------------------------------

If A, B, C take a, b, c days respectively,

Total 1 day work

    = 1/a + 1/b + 1/c


Time taken together

    = 1 / (1/a + 1/b + 1/c)


------------------------------------------------------------

4) WORK AND WAGES
-----------------

Wages are directly proportional to work done.

If A : B work ratio is

    1/a : 1/b

Then wage ratio is

    b : a


------------------------------------------------------------

5) EFFICIENCY CONCEPT
---------------------

Efficiency is inversely proportional to time.

If A takes 10 days
and B takes 20 days,

Efficiency ratio

    = 20 : 10
    = 2 : 1


If efficiency ratio is m : n,

Time ratio

    = n : m


------------------------------------------------------------

6) PIPE AND CISTERN
-------------------

Filling pipe (takes a days)

    1 day work = 1/a


Emptying pipe (takes b days)

    1 day work = −1/b


Net 1 day work

    = 1/a − 1/b


Time taken

    = 1 / (1/a − 1/b)


------------------------------------------------------------
