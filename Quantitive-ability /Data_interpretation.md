DATA INTERPRETATION
===================

------------------------------------------------------------

BASIC IDEA
----------

Data Interpretation (DI) involves

    • Reading data
    • Analyzing data
    • Applying arithmetic formulas
    • Drawing conclusions


Common Types of DI:

    • Bar Graph
    • Pie Chart
    • Line Graph
    • Table
    • Mixed Graph


------------------------------------------------------------

1) PERCENTAGE FORMULA (VERY IMPORTANT)
--------------------------------------

Percentage

    = (Value / Total) × 100


Percentage Increase

    = (New − Old)
      -------------
        Old
      × 100


------------------------------------------------------------

2) RATIO FORMULA
----------------

If two quantities are A and B,

Ratio

    = A : B


To compare,

    A / B


------------------------------------------------------------

3) AVERAGE FORMULA
------------------

Average

    = Sum of Values
      ----------------
      Number of Values


Total Sum

    = Average × Number


------------------------------------------------------------

4) PROFIT & LOSS (USED IN DI)
-----------------------------

Profit

    = SP − CP


Profit %

    = Profit / CP × 100


Loss %

    = Loss / CP × 100


------------------------------------------------------------

5) PIE CHART CALCULATIONS
-------------------------

Angle of Sector

    = (Value / Total) × 360°


Value from Angle

    = (Angle / 360) × Total


------------------------------------------------------------

6) GROWTH RATE (YEARLY DATA)
----------------------------

Growth %

    = (Present − Previous)
      -----------------------
        Previous
      × 100


------------------------------------------------------------

7) APPROXIMATION TECHNIQUE
--------------------------

In exams,

    • Round numbers for faster solving
    • Compare options before exact solving
    • Use percentage shortcuts


------------------------------------------------------------

8) DATA COMPARISON
------------------

Difference

    = Larger − Smaller


Percentage Difference

    = (Difference / Base) × 100


------------------------------------------------------------

9) TREND ANALYSIS (LINE GRAPH)
------------------------------

If data is increasing steadily

    → Positive trend


If decreasing steadily

    → Negative trend


If fluctuating

    → Irregular trend
