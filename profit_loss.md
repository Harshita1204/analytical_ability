=====================================================
               PROFIT AND LOSS
 ====================================================

BASIC DEFINITIONS
------------------------------------------------------------

Cost Price (CP)
    = Price at which article is purchased

Selling Price (SP)
    = Price at which article is sold


------------------------------------------------------------

1) PROFIT
------------------------------------------------------------

If SP > CP,

    Profit

    = SP − CP


Profit Percentage

    = Profit / CP × 100


------------------------------------------------------------

2) LOSS
------------------------------------------------------------

If CP > SP,

    Loss

    = CP − SP


Loss Percentage

    = Loss / CP × 100


------------------------------------------------------------

3) SELLING PRICE FORMULAS
------------------------------------------------------------

If Profit % is given,

    SP

    = CP × (1 + Profit% / 100)


If Loss % is given,

    SP

    = CP × (1 − Loss% / 100)


------------------------------------------------------------

4) COST PRICE FORMULAS
------------------------------------------------------------

If Profit % and SP are given,

    CP

    = SP / (1 + Profit% / 100)


If Loss % and SP are given,

    CP

    = SP / (1 − Loss% / 100)


------------------------------------------------------------

5) WHEN TWO PROFIT PERCENTAGES ARE COMPARED
------------------------------------------------------------

If article is sold at x% profit and y% profit,

Difference in SP

    = CP × (x − y) / 100


------------------------------------------------------------

6) MARKED PRICE (MP) & DISCOUNT
------------------------------------------------------------

Marked Price (MP)
    = Printed Price

Discount

    = MP − SP


Discount %

    = Discount / MP × 100


If Discount % is given,

    SP

    = MP × (1 − Discount% / 100)


------------------------------------------------------------

7) SUCCESSIVE DISCOUNT
------------------------------------------------------------

If two discounts are a% and b%,

Net Discount

    = a + b − ab / 100


------------------------------------------------------------

8) FALSE WEIGHT (IMPORTANT)
------------------------------------------------------------

If a shopkeeper claims no profit
but uses false weight,

Actual Profit %

    = Error% / (100 − Error%) × 100


------------------------------------------------------------

9) INCREASE & DECREASE IN PRICE
------------------------------------------------------------

If CP increases by x% and SP remains same,

Loss %

    = x / (100 + x) × 100


If CP decreases by x% and SP remains same,

Profit %

    = x / (100 − x) × 100


------------------------------------------------------------

10) WHEN PROFIT = LOSS CASE
------------------------------------------------------------

If Profit % = Loss % = x,

Net Loss

    = x² / 100


------------------------------------------------------------

IMPORTANT SHORTCUTS
------------------------------------------------------------

Profit % always calculated on CP  
Discount % always calculated on MP  
Loss % always calculated on CP  

------------------------------------------------------------
