=================
CODING – DECODING
=================

------------------------------------------------------------

BASIC IDEA
----------

Coding–Decoding questions test:

    • Pattern recognition
    • Logical reasoning
    • Alphabet position knowledge
    • Number transformation


------------------------------------------------------------

1) ALPHABET POSITION
--------------------

A  =  1  
B  =  2  
C  =  3  
...  
Z  =  26


Reverse order:

Z  =  1  
Y  =  2  
X  =  3  
...  
A  =  26


------------------------------------------------------------

2) LETTER SHIFTING
------------------

If each letter is shifted by +n positions,

New Position

    = Old Position + n


If shifting backward,

New Position

    = Old Position − n


------------------------------------------------------------

3) OPPOSITE LETTER (MIRROR CODE)
---------------------------------

Mirror Position

    = 27 − Alphabet Position


Example:

A (1)

    = 27 − 1
    = 26 → Z


------------------------------------------------------------

4) WORD VALUE METHOD
--------------------

Word Value

    = Sum of Alphabet Positions


Example:

CAT

C = 3  
A = 1  
T = 20  

Word Value

    = 3 + 1 + 20
    = 24


------------------------------------------------------------

5) NUMBER CODING
----------------

If numbers are transformed,

Look for:

    • Addition pattern
    • Multiplication pattern
    • Square or cube pattern
    • Difference pattern


------------------------------------------------------------

6) LETTER TO NUMBER REPLACEMENT
--------------------------------

Each letter may be replaced by:

    • Its position
    • Position + constant
    • Position × constant


------------------------------------------------------------

7) SERIES TYPE CODING
---------------------

Check for:

    • Alternate pattern
    • Increasing difference
    • Repeating cycle


------------------------------------------------------------

8) SYMBOL BASED CODING
----------------------

If symbols replace letters,

Step 1 → Decode meaning of each symbol  
Step 2 → Replace symbols with correct letters  
Step 3 → Form required answer


------------------------------------------------------------

9) IMPORTANT FORMULAS
---------------------

Alphabet Position

    = Letter Number


Reverse Position

    = 27 − Position


Word Sum

    = Sum of all letter positions


------------------------------------------------------------
