=====================
SERIES & PATTERNS
=====================

------------------------------------------------------------

BASIC IDEA
----------

Series & Pattern questions test:

    • Observation skill
    • Pattern recognition
    • Logical progression
    • Numerical reasoning


------------------------------------------------------------

1) NUMBER SERIES
----------------

Look for patterns like:

    • Addition
    • Subtraction
    • Multiplication
    • Division
    • Squares / Cubes
    • Prime numbers


Example Patterns:

Constant Difference

    2, 5, 8, 11, ...

    +3 each time


Increasing Difference

    2, 6, 12, 20, ...

    +4, +6, +8 ...


Multiplication Pattern

    3, 9, 27, 81, ...

    ×3 each time


Square Pattern

    1, 4, 9, 16, ...

    n²


------------------------------------------------------------

2) ALPHABET SERIES
------------------

Check:

    • Alphabet positions
    • +1, +2 shift
    • Alternate skipping


Example:

A, C, E, G, ...

    +2 shift


Position Method:

A = 1
B = 2
...
Z = 26


------------------------------------------------------------

3) MIXED SERIES
---------------

Combination of:

    • Number + Alphabet
    • Addition + Multiplication
    • Alternating patterns


Example:

2, 5, 4, 7, 6, 9, ...

Pattern:

    +3, −1 repeating


------------------------------------------------------------

4) ODD TERM IN SERIES
---------------------

Steps:

    Step 1 → Identify pattern
    Step 2 → Check which term breaks pattern


------------------------------------------------------------

5) SYMBOL SERIES
----------------

Look for:

    • Repeating cycle
    • Alternate pattern
    • Symmetry


------------------------------------------------------------

6) IMPORTANT SHORTCUT METHOD
-----------------------------

First check:

    • Difference between terms
    • Ratio between terms
    • Square / cube relation


If not clear:

    Check alternate terms separately
