=========================
DATA SUFFICIENCY
=========================

------------------------------------------------------------

BASIC IDEA
----------

Data Sufficiency questions test:

    • Logical judgment
    • Information analysis
    • Decision-making ability
    • Not full solving, but checking sufficiency


Goal:

    Determine whether given statements
    are sufficient to answer the question.


------------------------------------------------------------

1) STANDARD ANSWER FORMAT
-------------------------

Option A

    Statement I alone is sufficient


Option B

    Statement II alone is sufficient


Option C

    Both I and II together are sufficient


Option D

    Neither I nor II is sufficient


Option E (sometimes)

    Either I or II alone is sufficient


------------------------------------------------------------

2) IMPORTANT RULE
-----------------

Do NOT fully solve the question.

Only check:

    Can we determine a UNIQUE answer?


If answer is not unique,

    Data is NOT sufficient.


------------------------------------------------------------

3) TYPES OF QUESTIONS
---------------------

• Mathematical equations  
• Age problems  
• Ratio problems  
• Geometry problems  
• Number property problems  


------------------------------------------------------------

4) SOLVING STRATEGY
-------------------

Step 1 → Read question carefully  
Step 2 → Check Statement I alone  
Step 3 → Check Statement II alone  
Step 4 → Combine if needed  
Step 5 → Verify uniqueness  


------------------------------------------------------------

5) IMPORTANT CONCEPT
--------------------

If question asks:

    "What is the value of x?"

Then,

    Exact single value required.


If question asks:

    "Is x greater than y?"

Then,

    Clear YES or NO required.


------------------------------------------------------------

6) COMMON TRAPS
---------------

• Solving completely instead of checking sufficiency  
• Assuming extra data  
• Ignoring uniqueness condition  
• Mixing both statements too early
