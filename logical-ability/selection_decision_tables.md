==============================
SELECTION DECISION TABLES
==============================

------------------------------------------------------------

BASIC IDEA
----------

Selection Decision Table questions test:

    • Rule application
    • Condition checking
    • Logical filtering
    • Decision accuracy


You are given:

    • A table of conditions
    • Some eligibility rules
    • A candidate’s data

Goal:

    Decide the correct action.


------------------------------------------------------------

1) STRUCTURE OF QUESTION
------------------------

Usually includes:

    Condition 1
    Condition 2
    Condition 3
    Condition 4


And options like:

    (A) Selected
    (B) Rejected
    (C) Send for approval
    (D) Insufficient data


------------------------------------------------------------

2) TYPES OF CONDITIONS
----------------------

• Age limit  
• Qualification  
• Experience  
• Income  
• Marks percentage  
• Height / Weight  
• Work experience  


------------------------------------------------------------

3) IMPORTANT RULE
-----------------

If candidate satisfies ALL required conditions

    → Select


If candidate fails any mandatory condition

    → Reject


If candidate meets most but needs authority decision

    → Send for approval


If information missing

    → Insufficient data


------------------------------------------------------------

4) SOLVING STRATEGY
-------------------

Step 1 → Read all rules carefully  
Step 2 → Note mandatory conditions  
Step 3 → Check candidate data one by one  
Step 4 → Do not assume missing values  
Step 5 → Match with correct option  


------------------------------------------------------------

5) COMMON TRAPS
---------------

• Ignoring boundary values  
• Assuming missing data  
• Misreading “at least” or “not more than”  
• Forgetting special approval rule  


------------------------------------------------------------

6) IMPORTANT TERMS
------------------

At least

    ≥


At most

    ≤


More than

    >


Less than

    <
