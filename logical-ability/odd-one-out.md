=================
ODD ONE OUT
=================

------------------------------------------------------------

BASIC IDEA
----------

Odd One Out questions test:

    • Pattern recognition
    • Logical classification
    • Numerical observation
    • Concept grouping


Goal:

    Identify the element that does NOT follow the common pattern.


------------------------------------------------------------

1) NUMBER BASED ODD ONE OUT
----------------------------

Check for:

    • Prime numbers
    • Even / Odd pattern
    • Square / Cube numbers
    • Divisibility rule
    • Difference pattern


Example check methods:

    a) Common factor
    b) Sum of digits
    c) Multiplication relation
    d) Consecutive pattern


------------------------------------------------------------

2) ALPHABET BASED ODD ONE OUT
------------------------------

Check for:

    • Alphabet positions
    • Vowels / Consonants
    • +1 / +2 shift
    • Reverse order
    • Word value (sum of positions)


Alphabet positions:

    A = 1
    B = 2
    ...
    Z = 26


------------------------------------------------------------

3) WORD BASED ODD ONE OUT
--------------------------

Check for:

    • Category (Animal, Fruit, City, etc.)
    • Gender classification
    • Profession vs Object
    • Living vs Non-living


------------------------------------------------------------

4) SYMBOL OR MIXED TYPE
-----------------------

Check for:

    • Pattern repetition
    • Sequence break
    • Logical grouping


------------------------------------------------------------

5) SOLVING STRATEGY
-------------------

Step 1 → Identify common property  
Step 2 → Check each option  
Step 3 → Find one that breaks pattern  
Step 4 → Verify with at least two checks
