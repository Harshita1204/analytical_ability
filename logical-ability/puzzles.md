=================
PUZZLES
=================

------------------------------------------------------------

BASIC IDEA
----------

Puzzle questions test:

    • Logical linking
    • Arrangement skills
    • Data organization
    • Multi-condition reasoning


Most puzzles combine:

    • Seating Arrangement
    • Floor Arrangement
    • Ranking
    • Scheduling
    • Matching data sets


------------------------------------------------------------

1) TYPES OF PUZZLES
-------------------

1) Linear Puzzle  
2) Circular Puzzle  
3) Floor Based Puzzle  
4) Box / Scheduling Puzzle  
5) Matching (Person–City–Profession)


------------------------------------------------------------

2) FLOOR BASED PUZZLE
---------------------

If building has n floors,

Top Floor

    = Highest number


Ground Floor

    = Lowest number


Count floors carefully:

    Above
    Below


------------------------------------------------------------

3) MATCHING TYPE PUZZLE
-----------------------

If 3 categories are given:

    Person
    City
    Profession


Make table like:

    Person | City | Profession


Mark:

    ✔ Confirmed  
    ✘ Not possible  


------------------------------------------------------------

4) RANKING PUZZLE
-----------------

If position from top and bottom given:

Total Persons

    = Top Rank + Bottom Rank − 1


------------------------------------------------------------

5) IMPORTANT KEYWORDS
---------------------

Immediately

    = Next to


Between

    = One or more persons inside


Not

    = Eliminate possibility


Only

    = Fixed condition


------------------------------------------------------------

6) SOLVING STRATEGY
-------------------

Step 1 → Read full question once  
Step 2 → Identify categories  
Step 3 → Draw structured table  
Step 4 → Place definite clues first  
Step 5 → Use elimination method  


------------------------------------------------------------

7) COMMON TRAPS
---------------

• Ignoring “not” condition  
• Missing floor numbering direction  
• Assuming extra data  
• Not checking all conditions together
