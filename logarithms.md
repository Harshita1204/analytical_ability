==========
LOGARITHMS
==========

------------------------------------------------------------

BASIC DEFINITION
----------------

If

    a^x = N

then,

    logₐ N = x


Where,

    a = Base
    N = Number
    x = Logarithm


------------------------------------------------------------

COMMON LOGARITHM
----------------

Base 10 logarithm

    log N

means

    log₁₀ N


------------------------------------------------------------

NATURAL LOGARITHM
-----------------

Base e logarithm

    ln N

means

    logₑ N


------------------------------------------------------------

IMPORTANT LOGARITHM LAWS
------------------------

1) Product Law

    logₐ (MN)

    = logₐ M + logₐ N


------------------------------------------------------------

2) Division Law

    logₐ (M / N)

    = logₐ M − logₐ N


------------------------------------------------------------

3) Power Law

    logₐ (M^k)

    = k logₐ M


------------------------------------------------------------

4) Change of Base Formula

    logₐ N

    = log_b N
      ----------
      log_b a


Common form:

    logₐ N

    = log N
      -------
      log a


------------------------------------------------------------

IMPORTANT VALUES
----------------

log₁₀ 1

    = 0


log₁₀ 10

    = 1


logₐ a

    = 1


logₐ 1

    = 0


------------------------------------------------------------

CHARACTERISTIC AND MANTISSA
---------------------------

For common logarithm:

    log N

    = Characteristic + Mantissa


Characteristic

    = Integer part


Mantissa

    = Decimal part


If number > 1

    Characteristic = (Number of digits − 1)


If number < 1

    Characteristic = − (Number of zeros after decimal + 1)


------------------------------------------------------------

IMPORTANT SHORTCUTS
-------------------

If

    log a = x
    log b = y


Then,

    log (ab)

    = x + y


    log (a/b)

    = x − y


    log (a^n)

    = n x


------------------------------------------------------------

SOLVING LOG EQUATIONS
---------------------

If

    logₐ x = logₐ y

Then,

    x = y


If

    logₐ x = b

Then,

    x = a^b


------------------------------------------------------------

IMPORTANT SHORT NOTES
---------------------

• Log converts multiplication into addition  
• Log converts division into subtraction  
• Log converts power into multiplication  
• Always check base before solving  
• logₐ a = 1 is very important
